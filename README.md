## 安装

```
npm install --save react-stone
```

### 开始使用

```js
// 引入你所需要的组件
import { Button } from 'react-stone';
// 引入样式文件
import 'react-stone/build/static/css/main.css';

// 使用你所引入的组件
<Button className="delPic" icon="delete"></Button>

```
