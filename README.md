> A components library for React. 一个基于 React 的PC组件库


## 说明

目前这个组件库还处于开发阶段，组件数量比较少，欢迎大家提建议，贡献组件！/:rose


## 安装

```
npm install --save ceshi-stone
```

### 开始使用

Import ceshi-stone

```js
import { Button } from 'ceshi-stone';

<Button className="delPic" icon="delete"></Button>

```
